FavLinks

Introduction

Welcome to FavLinks! This application allows users to save and manage their favorite links in one place, providing an organized and efficient way to keep track of valuable online resources.

Features

  • Add, edit, and delete favorite links
  • Search functionality to easily find links
  • User authentication and authorization

Technology Stack

This project is built using the following technologies:

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Django
  • Database: SQLite
  • Hosting: PythonAnywhere

Usage Instructions

To get started with FavLinks:

  1. Register for a new account or log in if you already have one.
  2. Start adding your favorite links by clicking on the 'Add Link' button.
  3. Organize your links into folders for better management.
  4. Use the search bar to quickly find the link you need.
